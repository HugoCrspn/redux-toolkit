## Getting started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
```
git clone https://github.com/HugoCrspn/redux-toolkit.git
```

2. Install dependancies
```
npm install
```

3. Start app
```
npm start
```
